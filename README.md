This is demo project for testing zakaz.ua api (Service of delivery food in Ukraine)

Projetc is build on combination Playwright + Typescript + ZOD (Library for json schema validation)

For using:

1) Clone project

2) Create .env file inside project root derectory. Add env variables USER_EMAIL, USER_PHONE, USER_PASSWORD (need test account)

3) Launch tests via npx playwright test --grep '@api'

! Tag you can find in test description
